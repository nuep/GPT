# GPT Repository Overview

Witamy w repozytorium **GPT**. To repozytorium jest obecnie szkieletem przeznaczonym do eksperymentów i demonstracji związanych z modelami językowymi.

## Najważniejsze informacje

- Repozytorium startuje w bardzo lekkiej formie – zawiera jedynie pliki kontrolne Git oraz dokumentację.
- W katalogu `docs/` znajdziesz pogłębione materiały dotyczące proponowanej struktury projektu, standardów pracy i dalszych kroków.
- Wraz z rozwojem projektu warto utrzymywać przejrzystą strukturę katalogów (`src/`, `tests/`, `scripts/`, itp.), aby ułatwić nawigację i współpracę.

## Kolejne kroki

1. Zapoznaj się z dokumentem [`docs/overview.md`](docs/overview.md), który opisuje sugerowaną strukturę oraz dobre praktyki.
2. Ustal wspólne standardy dla zespołu (styl kodu, konwencja commitów, narzędzia CI) i odnotuj je w repozytorium.
3. Dodawaj nowe moduły w sposób iteracyjny, dbając o testy i dokumentację.

Powodzenia w rozwijaniu projektu!
